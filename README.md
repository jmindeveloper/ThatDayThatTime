## 프로젝트 소개

<br>

### 그날 그시간 - 시간의 기록
- 시간단위로 하는 기록
- 하루단위로 하는 기록
- 작성한 기록 검색
- 작성한 기록 월별로 모아보기
- iCloud 백업/복구
- 잠금모드 및 생체인식 잠금
- 기록을 까먹지 않도록 알림
- 다양한 글씨체

<br>
